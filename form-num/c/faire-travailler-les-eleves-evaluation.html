Utiliser le numérique en philosophie

g) Évaluer les élèves

Trouver un sujet

  • Rechercher un sujet dans la base de données des sujets du baccalauréat : dissertations 🔗 et explications de texte 🔗
  • Rechercher un sujet de dissertation dans une base de données générale (tous niveaux confondus) 🔗

Faire une dissertation ou une explication avec un dossier d'approfondissement

  • Partager des dossiers avec l'ENT ou les publier en ligne 🔗
  • Collaborer avec des collègues pour faire un dossier 🔗

Calculer le temps nécessaire pour une évaluation

  • Utiliser un outil qui compte les mots pour déterminer le temps de lecture d'un texte ou d'une consigne, ou le temps d'écriture pour un certain nombre de mots attendu 🔗

Annoter les copies

Utiliser des grilles

  • Utiliser une grille d'évaluation 🔗 🔗 🔗
  • Utiliser une grille pour repérer les problèmes récurrents ou des pistes pertinentes dans un tas de copies 🔗

Annoter numériquement des copies physiques

  • Utiliser un modèle de document pour ses corrections 🔗
  • Insérer des annotations prédéfinies 🔗, dans un navigateur 🔗 🔗 🔗, avec LibreOffice 🔗, avec Word 🔗
  • Utiliser un formulaire pour corriger 🔗
  • Faire des annotations vocales 🔗

Annoter numériquement des copies numériques

  • Utiliser la fonction “commentaire” dans un traitement de texte
  • Utiliser les fonctions d’annotation d’un fichier PDF 🔗
  • Utiliser un outil dédié pour corriger des copies 🔗

Vérifier si l'élève a fait un plagiat

  • Utiliser un moteur de recherche ordinaire et quelques astuces
  • Utiliser un logiciel anti-plagiat 🔗

Noter les copies

  • Utiliser Pronote pour différer la diffusion des notes et focaliser les élèves sur les annotations lorsque l’on rend les copies 🔗
  • Transférer ses notes rapidement d'un tableur vers Pronote 🔗
  • Utiliser un outil d’aide à l’évaluation 🔗

Rendre les copies

  • Utiliser l'ENT pour rendre les copies 🔗

Évaluer avec des QCM

  • Faire des QCM en philosophie ? 🔗 🔗 🔗
  • Utiliser un système de correction automatique de QCM 🔗 🔗 🔗
  • Faire des QCM avec Pronote 🔗 ou avec l'ENT 🔗, ou avec Google Forms 🔗
  • Intégrer des QCM dans le cours pour tester la compréhension du cours : en les intégrant dans un diaporama, avec réponses à l'oral, via des posts-it repositionnables 🔗, par téléphone 🔗, par papier codé 🔗 🔗, avec un boîtier de vote 🔗
